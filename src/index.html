About

This single-page website showcases common UI patterns: a sticky, resizing navbar with position indicator, smooth scrolling, a carousel, a fixed-background section, a modal, an embedded video, and responsive layouts.

Features

Responsive design ensures the layout adapts across common screen sizes. The navigation bar resizes as you scroll down. A scroll spy highlights your current section in the navbar.

The carousel demonstrates a simple slider with smooth transitions and accessible controls. The modal showcases layered UI and focus management. A fixed background section provides visual depth without heavy imagery or frameworks.

All styles are written in SCSS with variables, mixins, and keyframe animations. JavaScript is modular and minimal, using modern browser APIs.
